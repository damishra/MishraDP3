# Server Programming Project 3

## How to use

1. Clone repository :

```bash
    git clone https://github.com/MishraDP3.git
```

2. cd into repo directory :

```
    cd MishraDP
```

3. Install node dependencies :

```
    npm install && npm install companydata --save
```

4. Run the server :

```
    node server.js
```
